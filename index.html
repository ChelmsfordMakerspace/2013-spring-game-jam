What Chelmsford Game Jam?

Chelmsford Game Jam is an event where individuals or teams get one weekend to build a small game or prototype.

Guidelines

Within reason, you can use any language, game engine or framework you like to build your prototype, you can also built it for any platform or input method (Kinect, Playstation Move, Wii Remote, Balance Board, etc) you like as long as you can demonstrate it.

As well as a standard "Best in show" award, judges will also be looking to award strange and new ideas in made up categories, some suggested ones are listed below

  • Accessable gaming
  • New Input Methods

Semi-legal stuff

  • Attendees keep all intellectual property they create.
  • While it is not in any way required, we would prefer projects to have a public demo or be open-sourced.
